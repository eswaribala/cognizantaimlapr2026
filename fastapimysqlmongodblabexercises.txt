Lab 1: Customer Registration System

Business Scenario:
An e-commerce company wants to register customers and store structured customer details.

Database: MySQL

Tasks:

Create customers table.
Build FastAPI POST API to register customer.
Validate email and password using Pydantic.
Create GET API to fetch customer by ID.
Create GET API to list all customers.

Tables:

customers(
  customer_id,
  first_name,
  last_name,
  email,
  password,
  created_at
)

APIs:

POST /customers
GET /customers
GET /customers/{customer_id}
